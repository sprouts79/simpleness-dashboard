01 Meta Business Manager business.facebook.com
a
Opprett en ny Meta App
Gå til developers.facebook.com/apps/create
Velg type: Business → Koble til Simpleness sin Business Manager → Fullfør.
b
Opprett en System User
Business Settings → Users → System Users → Add
Navn: Simpleness Dashboard · Rolle: Admin
c
Gi System User tilgang til alle kundekontoer
Business Settings → Accounts → Ad Accounts
For hver kundekonto: Assign People → legg til "Simpleness Dashboard" → Rolle: Advertiser
d
Generer Access Token
System Users → Simpleness Dashboard → Generate New Token
Velg appen du opprettet → Huk av: ads_read, ads_management, business_management, read_insights
Kopier tokenet — det vises kun én gang.
Send til Jonas
Access Token (langt token som starter med EAA...)
App ID og App Secret (under App Settings → Basic)
Ad Account ID for hver kunde (format: act_1234567890)
02 GitHub github.com/new
a
Opprett et nytt repository
Gå til github.com/new
Navn: simpleness-dashboard · Sett til Private · Create repository.
b
Lag en Personal Access Token
github.com/settings/tokens/new
Navn: Claude Dashboard · Expiration: No expiration
Huk av: repo og workflow → Generate token.
Send til Jonas
Personal Access Token (starter med ghp_...)
Full repo-URL (github.com/brukernavn/simpleness-dashboard)
03 Vercel vercel.com/new
a
Opprett et nytt prosjekt
Gå til vercel.com/new
Import GitHub-repoet du nettopp opprettet → Framework: Next.js → Deploy.
(Første deploy feiler — det er ok, vi setter opp resten etterpå.)
b
Lag en API Token
vercel.com/account/tokens
Create Token → Navn: Claude Dashboard Deploy · Scope: Full Account · No Expiration.
Send til Jonas
Vercel API Token
Team ID (Vercel → Settings → General → Team ID)
Project ID (Vercel → Prosjektet → Settings → General → Project ID)
