Å legge til nye ad accounts til System User-en i Meta BM må gjøres manuelt én konto om gangen via business.facebook.com/settings/system-users. Jonas mener dette kan løses bedre via Meta's app-grensesnitt som støtter bulk-tildeling.

**Why:** Unødvendig manuelt arbeid ved onboarding av nye kunder.

**How to apply:** Når vi skal forbedre onboarding-flyten, se på om Meta Marketing API eller Meta Business Suite-appen har bulk-tildelingsendepunkter for system users.
